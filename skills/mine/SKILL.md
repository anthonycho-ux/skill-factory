# /mine — Make It Sound Like Mine

## Trigger

Invoke when:
- User says "proofread", "polish", "clean this up", "make this sound human"
- Text was generated by any AI model and needs a human pass
- Text needs to sound like a specific person's voice — not a model's approximation
- Writing is about to be published or shared

## What It Does

Takes AI-generated text and strips the model texture until it sounds like a person wrote it. Specifically *you*. Not rewriting — reclaiming the voice:

- **Taste**: your personal standard, not the model's defaults
- **Rhythm**: sentences that sound like how you actually speak
- **Typography**: the small things that feel off — em-dashes, spacing, punctuation habits
- **Structural**: parallel structure, missing connectors, buried clauses
- **Artifact removal**: Chinese-origin character artifacts, UTF-8 noise, unnatural phrasing left by the originating model

## The Insight

Every AI model has a fingerprint. Minimax M2.7 leaves subtle rhythm issues and character artifacts. Claude produces clean grammar but flat cadence. Gemini sounds polished but generic. None of them sound like *you*.

The move: two-model stack.
1. First model generates — fast, creative, unconstrained
2. Second model polishes — local, fast, tuned to your taste

Gemma E2B via Ollama catches what the first model left behind — because it's reading for *your* audience, with *your* expectations. Not the model's.

The goal is not perfect grammar. The goal is **soul** — the part of the writing that is unmistakably human, unmistakably yours.

## Workflow

```
1. Identify the text (file path, pasted content, or AI-generated draft)
2. Name the originating model (e.g. "this came from Minimax M2.7")
3. Load Gemma E2B via Ollama
4. Call /api/chat (NOT /api/generate — generate endpoint silently
   truncates gemma4:e2b responses at high token counts):

   curl -s http://localhost:11434/api/chat -d '{
     "model": "gemma4:e2b",
     "messages": [
       {"role": "system", "content": "You are a taste filter. Read this text and fix only the things that make it sound AI-generated: typos, rhythm issues, structural awkwardness, character artifacts, flat parallel structure. Do NOT rewrite. Do NOT restructure arguments. Preserve the author's voice — only remove the seams. Return only the corrected text."},
       {"role": "user", "content": "[YOUR TEXT HERE]"}
     ],
     "stream": false
   }'

5. Extract response from: .message.content in the JSON response
6. Diff: list exactly what changed and why
7. Present diff to user for approval
8. On approval — save, overwrite, or suffix "_mine.md"
```

## Rules

- **Preserve soul.** Don't sand down the personality to make it "correct." If the voice is strange in an interesting way, keep it.
- **Never rewrite.** This is not an editing pass. It's a reclaiming pass. Don't add, don't restructure, don't improve — only remove what's AI-generated.
- **Show the diff.** Always list what changed. The diff is how you train your taste over time.
- **Name the source model.** Every model has different tells. Tracking the source sharpens calibration.
- **Local first.** Use Ollama. No API calls for `/mine` — it's fast, private, and yours.
- **Light touch.** A human doesn't rewrite. A human reads and adjusts. That's what this skill does.

## Examples

**Input** (Minimax M2.7 — structural fragmentation):
```
I'd re-solve the same problem. Miss the same insight. Arrive at the same dead end I'd visited three months ago and completely forgotten.
```

**Gemma corrects to:**
```
I'd re-solve the same problem, miss the same insight, and arrive at the same dead end I'd visited three months ago and completely forgotten.
```

**Diff:**
- Collapsed three staccato sentences into parallel structure with commas
- Voice preserved — the rhythm of someone recounting a pattern

---

**Input** (missing conjunction — fragment):
```
Most AI skills are magic spells: you say the right words, something happens.
```

**Gemma corrects to:**
```
Most AI skills are magic spells: you say the right words, and something happens.
```

**Diff:**
- Added missing "and" — the sentence was a fragment without a conjunction
- Nothing else touched

---

**Input** (flat cadence):
```
I learned something. Then I forgot it. Then I re-learned it. This happened many times.
```

**Gemma corrects to:**
```
I learned something. Forgot it. Re-learned it. This happened so many times I stopped counting.
```

**Diff:**
- Shortened sentences for rhythm — the staccato is intentional here
- "many times" → "so many times I stopped counting" — more human, more honest

## Dependencies

- Ollama running locally
- Gemma E2B loaded (`ollama pull gemma4:e2b` if missing)
- Network access NOT required — this runs fully offline
- **Important:** Use `/api/chat` endpoint — `/api/generate` silently truncates gemma4:e2b responses at high token counts

## Calibration

The skill compounds. Track it:

- **Gemma catches, user fixes manually** → Gemma needs a better system prompt
- **Gemma changes, user reverts** → add that pattern to "do not touch"
- **User finds something Gemma missed** → add to Gemma's system prompt
- **Different source model** → note the new model's tells separately

Over time, Gemma learns your palate. What starts as a tool becomes an extension of your sensibility.

## The Deeper Intention

`/mine` is not about proofreading. It's about the realm AI cannot touch: the specific rhythm of your thoughts, the way you phrase something that no model would invent, the thing you're actually trying to say.

The skill removes what AI added. What remains is what you meant.
