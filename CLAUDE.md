# Skill Factory

A workshop for designing, building, and shipping OMC skills and personas — fast, with cognitive science as the foundation.

## Mission

Turn hard-won knowledge (from research, mistakes, workflows) into reusable, evidence-based skills that improve every session.

## Workflow

1. **Discover** — Identify a gap or repeated friction in daily use
2. **Research** — Ground the skill in cognitive science or domain best practices
3. **Draft** — Write SKILL.md with trigger, behavior, workflow, examples
4. **Test** — Use the skill in real sessions, iterate based on feedback
5. **Ship** — Push to `anthonycho-ux/sovereign-omc-skills` and update install docs
